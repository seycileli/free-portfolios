Get more done.

Keep track of your tasks and never miss another Due Date

ToDo Task manager built with using JavaFX. Your ToDo tasks are stored through an XML file, using a Data Binding technique that binds data sources from the provider along with consumer together and synchronizes them. In a Data Binding process, each changed data is reflected automatically by the User's input into the Data. Which is why in the demo above you can see with each new Creation, Edit or Deletion is acted upon in seconds.

Features:
Give the User an option to Create a New ToDo task from a drop down menu
Give the User an option to Create a New ToDo task by an easy to find button
Give the User the option to give their ToDo task a subject title
Give the User an option to attach a note to their ToDo task
Give the User an option to assign a Due Date to their ToDo task
Given that a User may not check the Due Date as often, create a user friendly system that will remind the User of their ToDo task Due Date
Give the User an option to delete their ToDo task from a drop down menu
Give the User an option to delete their ToDo task by right clicking
Make a save functionality for the User so that they don't lose track of any new created ToDo task or edited / updated tasks

As the users ToDo task deadline approaches, the user will be reminded by a color system.

Red: indicates that the ToDo Task is Due Day of (today)

Peach: indicates ToDo Task is Due in 1 day

Yellow: indicates ToDo Task is Due within 2 days

Light Green: indicates ToDo Task is Due within 4 days

Green: indicates ToDo Task is Due within 7 days or more

Tools Used: Java, JavaFX, XML, FXML
