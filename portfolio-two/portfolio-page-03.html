SchoolManagement System

Register before it's too late

The School year is near, register now before it's too late

School Management System where students can register to courses, and view the course assigned to them.

The task is to create a basic School Management System where students can register to courses, and view the course assigned to them.

Features:
Students can register to a course that they are not enrolled in
Students can view which courses they are enrolled in
Student information are saved as a Comma-Separated-File (CSV) file
Course information are saved as a CSV file
Attending CSV file holds record of which Student attending which Course

Tools Used: Java, CSV
