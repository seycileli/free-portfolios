My Contacts

Keep them close.

Your contacts stored securely, who are a phone call away.

My Contacts application is built using JavaFX. Your Contacts are stored through an XML file, using a Data Binding technique that binds data sources from the provider along with consumer together and synchronizes them.

In a Data Binding process, each changed data is reflected automatically by the User's input into the Data. Which is why in the demo above you can see with each new Contact Creation, Edit or Deletion is acted upon in seconds.

Features:
Give the User an option to Create a New Contact from drop down menu
Give the User an option to enter First / Last Name, Phone Number and a note
Give the User an option to Edit a Contact(s) information
Give the User an option to Delete a Contact(s) information
If a User wants to Edit / Delete a Contact but has no one selected, notify User with a pop up message

Tools Used: Java, JavaFX, XML, FXML
